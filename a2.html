Assignment 2: Fade

The goal of this assignment was to build and code a circuit with a minimum of 3 LEDs fading in or out, initiated by some sort of input.

Yellow & Red LEDs:
V = 5, Vdrop = 1.8, I = 20mA

3.2v = 20mA x Ω

Blue LED:

Final Circuit

Built circuit made as shown on the schematic.

Code

        
          //The analog PWM pin the first (red) LED is attached to
          const int led = 9;

          //The analog PWM pin the second (blue) LED is attached to
          const int led2 = 10;

          //The analog PWM pin the third (yellow) LED is attached to
          const int led3 = 11;

          //The pin the button is attached to
          const int buttonPin = 2;

          // Initializing the state of the button
          int buttonState = 0;


          void setup() {

            //Initializing all LED pins as outputs
            pinMode(led, OUTPUT);
            pinMode(led2, OUTPUT);
            pinMode(led3, OUTPUT);

            //Initializing the button pin as an input
            pinMode(buttonPin, INPUT);
          }

          void loop() {

            //Section to be executed when the button is pressed
            if (buttonState == HIGH) {

              //To loop through each of the pins one at a time
              for (int j = 9; j <= 11; j++) {

                //Loop to change the brightness one point at a time going from 0 -> 255
                for (int i = 0; i < 255; i++) {

                //Writes the value of i to the brightness of the pin for a fade in
                analogWrite(j, i);

                //Adds a delay of .1 ms between brightness changes
                delay(10);
                }

                //Loop to change the brightness one point at a time going from the
                // end of the last loop (255 -> 0)
                for (int i = 255; i > 0; i--) {

                //Writes the value of i as the brightness state of the LED
                analogWrite(j, i);

                //Adds a delay of .1 ms between brightness changes
                delay(10);
                }

                //Turns the LED off once the cycle has completed
                digitalWrite(j, LOW);

              }

            //Logic for when the button is not pressed
            } else {

              //Turns all of the LEDs off when the button is not pressed
              digitalWrite(led, LOW);
              digitalWrite(led2, LOW);
              digitalWrite(led3, LOW);
            }
          }

Circuit Function

The function of the completed circuit that has three LEDs fading in and out one at a time when input from a button is detected.
